## 📱 移动端适配详情

### 响应式断点
```
xs: < 576px    // 手机端 - 单列布局
sm: 576-768px   // 平板端 - 双列布局
md: > 768px     // 桌面端 - 多列布局
```

### 具体优化
1. **搜索表单**：从inline改为Grid布局
2. **表格滚动**：添加`scroll={{ x: 'max-content' }}`
3. **按钮组**：`Space wrap`属性自动换行
4. **卡片布局**：Col响应式宽度设置
