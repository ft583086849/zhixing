## 📝 使用说明

1. **管理员**: 先访问 `/admin` 用 admin/admin123 登录
2. **销售注册**: 访问 `/sales` 注册成为一级分销
3. **用户购买**: 访问 `/purchase` 进行购买
4. **对账查看**: 注册成功后可访问对应的结算页面

## 🔄 业务流程
1. 管理员登录 → 配置系统
2. 一级分销注册 → 获得推广链接
3. 二级分销注册 → 关联一级分销
4. 用户通过链接购买 → 佣金分配
5. 分销查看对账页面 → 佣金结算
