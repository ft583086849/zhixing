## 性能测试

### 1. 工具推荐
- Lighthouse
- WebPageTest
- GTmetrix
- PageSpeed Insights

### 2. 测试指标
- First Contentful Paint (FCP)
- Largest Contentful Paint (LCP)
- First Input Delay (FID)
- Cumulative Layout Shift (CLS)

### 3. 性能预算
- 页面加载 < 3秒
- API响应 < 500ms
- 图片加载 < 2秒
- 交互响应 < 100ms
